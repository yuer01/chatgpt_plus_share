* 2、【优化】`万能表格` `tag` 类型渲染支持 数组形式（渲染标签组），同时新增effect：dark / light / plain 3选1

```js
{ 
  key:"xxxxx", title:"身份类型", type:"tag", width:200,
  effect:"light",
  data:[
    { value:"region", label:"区域代理" },
    { value:"shareholder", label:"商城股东", tagType:"danger" }
  ]
}
```
